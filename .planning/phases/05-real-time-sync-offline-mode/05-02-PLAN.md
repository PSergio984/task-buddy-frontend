<task type="auto">
  <name>Task 2: Optimistic UI for Tasks & Subtasks</name>
  <files>src/hooks/useTasks.ts</files>
  <action>
    - Implement optimistic updates for `useCreateTask`, `useUpdateTask`, `useDeleteTask`, and `useToggleTask`.
    - Pattern: In `onMutate`, cancel outgoing queries, snapshot current data, and update the cache immediately with the new item (using a temp ID if necessary).
    - In `onError`, rollback to the snapshot.
    - In `onSettled`, invalidate the query to sync with server truth.
    - Repeat for subtask hooks (`useCreateSubtask`, `useUpdateSubtask`, etc.).
  </action>
  <verify>
    <automated>npm run lint</automated>
  </verify>
  <done>Task mutations feel instantaneous through optimistic UI.</done>
</task>
