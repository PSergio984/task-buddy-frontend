# Phase 4.2: High-Velocity Polish & Feature Hardening - Plan

This phase hardens the core task interactions, consolidates the navigation model into a "Drawer-Only" workflow, and improves visibility across the Dashboard and Tasks page.

## Success Criteria
1.  **Sidebar-Filter Sync**: Clicking sidebar smart lists (Inbox, Today, Upcoming) activates corresponding pills in the Tasks filtering hub.
2.  **Drawer-Only Creation**: Task creation moves from a modal to a consistent right-side drawer.
3.  **Subtask CRUD**: Subtasks are fully functional (Add, Toggle, Delete) within the Task Detail Drawer.
4.  **Notes Evolution**: Comments are replaced by an expanded, auto-saving Description/Notes area.
5.  **Dashboard Granularity**: A global timeframe switcher updates status widgets and task lists.
6.  **Safety & Performance**: Snappy drawer animations (<200ms) and confirmation modals for deletion.
7.  **Search-and-Create**: Project/Tag selectors support searchable lists and inline creation on "Enter".
