## Sidebar Controls

| Option | Description | Selected |
|--------|-------------|----------|
| Collapsible Sections | Add a chevron icon to Projects/Tags headers to hide/show the list items. | ✓ |
| Static Lists | Keep lists always visible, only collapsing when the entire sidebar collapses. | |
| Toggle Off | Clicking an active filter reverts the view to 'All Tasks'. | ✓ |
| Stay Active | Clicking an active filter does nothing. | |

**User's choice:** Collapsible Sections, Toggle Off
**Notes:** Users want independent control over sidebar list visibility and an intuitive way to clear filters.

## History Readability

| Option | Description | Selected |
|--------|-------------|----------|
| Natural Language | 'Changed [Field] to [New Value]' (shorter, cleaner). | |
| Full Diff | 'Updated [Field] from [Old] to [New]' (maximum transparency). | ✓ |
| Include Parent Name | 'Finished [Subtask Name] in [Parent Task Name]'. | ✓ |
| Subtask Only | 'Finished [Subtask Name]'. | |

**User's choice:** Full Diff, Include Parent Name
**Notes:** Users prefer detailed update logs over over-simplified ones, and need parent context for subtasks.

## Settings & Polish
